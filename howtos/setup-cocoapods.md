# How to setup / install cocopods

## Setup

```bash
> sudo gem install cocoapods
> cd <project>
> pod init
> vi Podfile
> pod install

```
Close project.
Reopen with workspace file.
Use.

How to search for pod from command line (better than web)

```bash
> pod repo update
> pod search firebase
```
